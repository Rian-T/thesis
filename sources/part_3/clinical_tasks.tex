The first two parts built a public biomedical corpus and pretrained encoders on
it. This part asks whether those public foundations can support a real clinical
task without any training on hospital data: reading a patient's free-text
reports and filling the structured form that a clinical study keeps for each
patient. The task is demanding because the same value plays different roles as
the disease unfolds, and because no real filled form is available to learn from.
The next three chapters build task supervision from public and generated text,
design an extractor that can be asked for new fields at inference time, and
compare a small biomedical encoder with much larger generative models. The
result they build toward is that a 150-million-parameter encoder, trained only
on public and generated text and never shown a real hospital record, nearly matches
the accuracy of a generative model more than twenty-five times its size
fine-tuned on the same data.
