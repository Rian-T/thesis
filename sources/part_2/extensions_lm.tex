A corpus is only useful once a model has learned from it. This part puts the
biomedical corpus to work and asks what makes pretraining on it effective. We first
continue the pretraining of an existing French encoder on biomedical text. We then
return to the changes that made recent decoder models stronger, and test which of
them can help an encoder. These changes concern modern architecture, causal
training, and better data selection. The final chapter adds another source of signal by turning structured
medical knowledge into text the model can read.
